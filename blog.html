The Document Object Model(DOM) is a programming interface for HTML Document. DOM represent your HTML page in such a way that it is easy to modify with programming language. Like document structure, style, and content. We need DOM to access or represent our HTML Document in an Object Oriented way. And we need a contention for DOM with HTML Document for access and this contention done with JavaScript language.

Select HTML elements using DOM as a programmer or developer we need to know how to select HTML element using DON. There are six types of methods to select an element from the document.
1. getElementById – for access ID.
2. getElementsByTagName – for access tag link div, header , footer ,section.
3. getElementsByClassName – for access class.
4. getElementsByName – for access name attribute.
5. querySelector – for selecting only the first element from the code.
6. querySelectorAll – for returns a static NodeList.
